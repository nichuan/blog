Java8中的字符串连接收集器

在JDK8中，可以采用函数式编程（使用 Collectors.joining 收集器）的方式对字符串进行更优雅的连接。
Collectors.joining 收集器 支持灵活的参数配置，可以指定字符串连接时的 分隔符，前缀 和 后缀 字符串。
